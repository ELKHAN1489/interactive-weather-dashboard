# Interactive Weather Dashboard 🌤️

Современный интерактивный погодный дашборд с динамическим интерфейсом, умным поиском и аналитическими графиками. 

## 🚀 Основной функционал
* **Геолокация в реальном времени:** Автоматическое определение местоположения пользователя через Browser Geolocation API при первом входе.
* **Умный поиск (Autocomplete):** Живой поиск городов с подсказками "на лету" через эндпоинт WeatherAPI Search, исключающий ошибки ввода.
* **Интерактивные графики:** Плавная визуализация изменения средней температуры на несколько дней вперед с помощью библиотеки **Chart.js**.
* **Динамический визуал:** Автоматическая смена фоновых изображений и иконок высокого разрешения в зависимости от текущих погодных условий (дождь, снег, ясно, облачно).
* **Премиальный интерфейс:** Адаптивный Glassmorphism-дизайн (эффект матового стекла), оптимизированный под любые экраны (от смартфонов до 2K мониторов).

## 🛠️ Стек технологий
* **Frontend:** HTML5, CSS3 (Grid, Flexbox, Webkit-filters, Адаптивные медиа-запросы).
* **JavaScript:** ES6+, Асинхронные запросы (Async/Await, Fetch API), динамическая манипуляция DOM.
* **Библиотеки:** Chart.js (Отрисовка графических данных), Boxicons (Пакет современных векторных иконок).
* **API:** WeatherAPI (Погодный дата-центр).
